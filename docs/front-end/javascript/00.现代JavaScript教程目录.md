## 第一部分：JavaScript 编程语言

在这儿我们将从头开始学习 JavaScript，也会学习 OOP 等相关高级概念。本教程专注于语言本身，默认使用最小环境。

### 章节概览

1. **简介** - JavaScript 简介、手册与规范、代码编辑器、开发者控制台
2. **JavaScript 基础知识** - 变量、数据类型、运算符、条件分支、循环、函数等
3. **代码质量** - 调试、代码风格、注释、自动化测试、Polyfill
4. **Object（对象）基础知识** - 对象、引用复制、垃圾回收、this、构造器等
5. **数据类型** - 数字、字符串、数组、Map/Set、解构赋值、JSON等
6. **函数进阶内容** - 递归、闭包、装饰器、箭头函数深入等
7. **对象属性配置** - 属性标志、getter/setter
8. **原型，继承** - 原型继承、原生原型、原型方法
9. **类** - Class语法、继承、静态属性、私有属性、instanceof
10. **错误处理** - try...catch、自定义Error
11. **Promise，async/await** - 回调、Promise、Promise链、微任务、async/await
12. **Generator，高级 iteration** - generator、异步迭代
13. **模块** - 模块简介、导出导入、动态导入
14. **杂项** - Proxy、Reflect、Eval、柯里化、BigInt、Unicode

## 第二部分：浏览器：文档，事件，接口

学习如何管理浏览器页面：添加元素，操纵元素的大小和位置，动态创建接口并与访问者互动。

### 章节概览

1. **Document** - DOM树、遍历DOM、搜索元素、节点属性、修改文档、样式和类
2. **事件简介** - 浏览器事件、冒泡和捕获、事件委托、自定义事件
3. **UI 事件** - 鼠标事件、键盘事件、滚动事件、拖放事件
4. **表单，控件** - 表单属性方法、聚焦、表单事件
5. **加载文档和其他资源** - 页面生命周期、async/defer脚本、资源加载
6. **杂项** - Mutation observer、Selection和Range、事件循环

## 📝 学习建议

1. **循序渐进**：按照教程顺序学习，不要跳跃
2. **动手实践**：每个概念都要自己写代码验证
3. **做习题**：完成每章后的练习题
4. **反复复习**：定期回顾已学内容
